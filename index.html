Como usar:

  1. Clique na área "Carregar Imagem" para selecionar uma imagem do seu dispositivo
  2. Aguarde até que a imagem seja carregada na pré-visualização
  3. Use os sliders para ajustar as posições das cores no mapa degradê
  4. Clique no botão "Redefinir Paleta" para voltar aos valores originais
  5. Clique no botão "Aplicar Paleta" para aplicar as cores especificadas
  6. Use o botão "Salvar" para baixar o resultado da alteração
  7. Você também pode arrastar e soltar uma imagem diretamente na área de upload
